CLONE ECOMMERCE JORDAN 
Autentificazione, JSON server e Fetch

Per aprire il progetto seguire i seguenti step: 

Sul terminale eseguire:

1 - npm i  (installare node modules)
2 - npm i json-server  (installare json server)
3 - json-server --watch assets/data/data.json (eseguire il comando per lanciare il db)
4 - lanciare l'app con LiveServer
